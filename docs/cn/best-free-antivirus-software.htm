简介

杀毒软件为计算机提供基础的防御层次，防止大量病毒、木马、蠕虫、间谍软件、广告软件、拨号器、键盘记录器和rootkit隐藏程序的感染。在过去，反病毒程序仅检测病毒，而反间谍软件仅检测间谍程序，但现在而言，杀毒软件和反间谍软件的界限分别不很清晰了。大多数现有的杀毒软件对于各种形式的恶意软件都至少有足够合理的检测率。

恶意软件包括病毒、木马、蠕虫、间谍软件、广告软件、拨号器、键盘记录器和rootkit隐藏程序等，在计算机上执行各种恶意活动。由于恶意软件数量急剧增加，杀毒软件不能够跟上脚步全部检测出来。即使利用多个实时杀毒软件，也会耗用更多系统资源，导致冲突并削弱防护水平。因此本人推荐您仅选择一种杀毒软件实时保护，另加其他的安全软件来提高防护水平。

我看了数款免费的杀毒程序，对于保护您的计算机免受病毒和其他恶意软件的威胁，具有至关重要的意义。
 
讨论

Avast!免费杀毒软件是我在考虑到最近的发展形势后，所认为的首选最佳免费杀毒软件中。Avast(英文维基百科)在过去的数年间一直稳定地提高它的检测率，并且现在已被列为最好的杀毒软件之一。

Avast 软件拥有一切免费杀毒软件的绝大多数功能，其全面实时的防护能力，包括网页、电子邮件、即时通讯、P2P和网络防护，启动时的扫描，以及一个行为拦截器。其新的第6版本还带来了更多的功能，包括互联网站点评级插件，阻止恶意脚本程序，以及一个功能有限的（不是可配置的）沙盒功能。虽然沙盒是一个非常有用的半自动安全功能，但是默认的设置包括了一个弹出窗口，来询问用户是否把一个软件放进沙盒运行——对于典型的普通用户而言，这可能会稍微让人不能明白问题的含义，但也能设置让程序自动把软件放进沙盒运行。最后，Avast的资源需求也较低。

微软安全套件 (Microsoft Security Essentials) 也是我最喜爱的免费杀毒软件之一，它有很高的检测率，特别是在对付rootkit隐藏程序时。更让人印象深刻的是微软安全套件(英文维基百科)几乎没有误报，资源消耗低，并且在移除现存的恶意软件时工作得很好。

MSE (微软安全套件)对于普通用户而言是一个极好的选择，因为它需要与用户交互的地方已经减少到最低限度。它会自动地更新并移除威胁。尽管它要快速地验证一下Windows是否正版，但它不需要注册，并且没有烦人的屏幕骚扰信息或广告。

其主要的缺点是缓慢的扫描速度和在隔离恶意软件时它会花费大量的时间，但在本人看来，这些只是微小的不足，因为举例来说，扫描可以在过夜时运行。在某些国家没有现成的MSE版本提供，因此那里的用户们不得不在别的地方找到它。要注意的是微软安全套件需要在正版的Windows上才能安装。

熊猫云引擎杀毒软件 (Panda Cloud Antivirus) 是熊猫公司进入免费杀毒领域的成功尝试。它也像MSE，对于普通用户而言是一个极佳的选择，它带有简洁的界面以及完全自动化的功能，包括自动更新和删除恶意软件。其检测水平通常是非常高的，但根据最新的测试数据，其检测零日威胁的水平稍稍低了点。PCA(英文维基百科)使用“云”技术来让更新发布快速得很多，因为所有的定义储存在云上，因此一旦熊猫公司更新了他们的黑名单，那么PCA所有有互联网连接的用户就会被保护以免受侵害。

熊猫云引擎杀毒软件带有行为拦截器和网页防护功能，这当然会提高你的安全水平。但是当您使用其他免费杀毒软件，并仅使用其所带的独立行为拦截器，而这又有可能提供更好的防护时，那么前者就未必是一个优势了。

我所知道的PCA的一个小小的不足，是它好像会错误地检测浏览器/系统相关的应用程序，例如 VideoCacheView 软件，并且由于它会自动进行隔离，这就会带来一些小麻烦。

小红伞杀毒软件个人版 (Avira AntiVir Personal Edition) 是另一款高质量的免费杀毒软件，并且以前是本文的首选。我将它的排位加以调整的主要原因，涉及到最近关于杀毒软件的争论，包括在它们的安装包中有一个重新改名包装的Ask工具条，以及临时的接二连三的广告，而这广告会推广一个可能与恐吓软件有关的公司，尽管有这样的负面新闻，AntiVir(英文维基百科)仍然是一个顶尖的反病毒扫描器，并且其对恶意软件的检测率极其出色，根据测试，其检测率高于几乎别的每款杀毒软件。特别是对于有重大风险的用户而言，AntiVir仍然是个不错的免费杀毒扫描器选择。

但是，它也有一些不足。首先，AntiVir不包括网页或电子邮件扫描功能；该功能仅出现于付费版本中。没有电子邮件扫描器并不是一个缺点，它仅意味着在您打开受感染的电子邮件之前，AntiVir不会给你发出警告。但如果你打开了一封受感染的电子邮件，AntiVir仍然会跳出来执行有关动作，所以它并不意味着你不会受到保护而受到基于电子邮件的感染。另外，AntiVir在每次更新时，会包含一个相当侵犯性的广告，虽然AntiVir在过去有签名更新问题，但现在该问题似乎已被修复了。

AVG 杀毒软件免费版也是一个体面的免费反病毒软件选择。虽然在最近的过去，这款可敬的杀毒软件在检测中有一些不稳定的结果，但它现在提供了一个高水平的防护。这可能是由于它包含了一个完全成熟的行为拦截器，即“Identity Protection”防护技术。这提供了一个非常重要的额外防护层，但是在另一方面，通过使用上述的免费反病毒软件以及功能独立的行为拦截器，也能达到类似防护水平。令人遗憾的是，AVG免费版在尺寸上增长得相当大，扫描速度非常慢，并且有广告（但可以禁用广告）。

这些免费的反病毒程序都是优秀的软件，它们为用户提供了真正可以替代主要商业反病毒产品的选择。
