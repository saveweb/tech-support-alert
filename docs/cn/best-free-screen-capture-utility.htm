简介

在标准的视窗操作系统中，按“Print-Screen”键，就会把整个屏幕的图像拷贝到剪贴板，在按住Alt键的同时按下“Print-Screen”键，就会捕获当前选定的窗口。Vista和Windows 7预装了一个截屏工具(译者注：Win7基础版中没有该工具)。其它操作系统也可能提供类似的截屏实用工具。

作为随操作系统而来工具的替代品，本评论中的免费软件产品涵盖了下列大多数有用的特性和功能，用户们也会发现确实需要这些功能。

  • 捕获全屏或自由选定的屏幕区域。
  • 自动探测窗口和控件对象，如按钮、输入框、工具条、选项卡等。
  • 照下滚动窗口，允许延时捕获。
  • 改变屏幕快照的图像尺寸，或者创建缩略图，以及其它附加功能。

本文的中心是关于屏幕捕获。如果你首要任务是在截屏后编辑图像，那么请阅读《最佳免费数字图像编辑器》。那里列出的若干个编辑器也有截屏功能。
 
讨论

Screenshot Captor支持通过热键或程序菜单捕捉多个屏幕、全屏、窗口、矩形区域和固定区域。可以包含也可以不包含鼠标光标。它是唯一一个能给自身截屏的截屏工具。该软件也能从扫描仪获取图像。

一旦捕获图像，该程序会自动保存图像、打印图像、把图像保存到剪贴板、保留图像、在内置编辑器中显示图像、通过电子邮件把图像发送出去，或者把图像上传到存取图像的服务网站上。

其内置的编辑器功能丰富。你可能会发现不必再使用一个图像编辑器。

它会弹出提示要求注册，但注册是免费的。

 

DuckCapture支持捕获某个区域、窗口或对象、滚动窗口以及整个屏幕。你可以使用热键或用鼠标点击系统托盘上的程序图标，来选择你要哪种捕获模式。

使用本程序，你可以挑选图像的轮廓线、设置输出目标到剪贴板或自动保存到某个文件。

我发现本程序直观易用，但它缺少一个简单的内置图像编辑器，并且没有允许延时捕获的选项。

 

SnapDraw是个截屏工具 - 编辑器 - 注释工具。它仅支持捕捉窗口或整个屏幕，但是可以用内置的编辑器对图像进行裁剪。它没有延时截屏的设置。可以通过热键或通知图标进行操作。

它的优秀之处在于其内置的编辑器。里面提供了许多工具，如标注、放大镜、3D透视图，以及反射功能(?)。

图像可以被保存到剪贴板、文件、电子邮件或FTP中。没有直接的打印机选项。

 

Lightscreen会通过热键或托盘图标来捕捉屏幕、窗口及区域。它支持延时快照、鼠标光标和多个显示器。它会自动把输出保存到文件或剪贴板。它不支持其它输出功能。

 

Cropper软件让用户能通过热键或通知图标，轻松快速地抓取屏幕的某些部分。它支持捕获整个屏幕、活动窗口和矩形区域。没有自动保存、自动滚动和延时抓取功能。它可以同时创建屏幕快照和缩略图。

其输出能被定向到剪贴板、打印机或某个图像文件。

 

在 Windows Vista 和 Windows 7 中已经安装了微软的截图工具(Snipping Tool)（译注：32位的中文家庭基础版中没有）。要首先启动该程序，才能开始截屏操作。用户可从程序中选择截屏类型或按下Escape键及PrintScreen热键。它支持截取全屏、活动屏幕、矩形区域和手工绘制区域。它没有自动保存、自动滚动或延时捕获功能。

输出结果会显示在编辑器窗口中，并且会用电子邮件发送出去或存为图像文件。编辑器仅限有两种工具——手工高亮突出工具和手工绘制直线工具。它没有打印机输出功能。

 

Screen Grab Pro的特色是从主界面或用热键一键抓取桌面、活动窗口和区域，允许定时操作，并把图像拷贝到剪贴板或外部编辑器。

我喜欢的是，它有个选项能在保存文件后自动打开外部编辑器。

当用户试图抓取某个矩形区域时，屏幕的分辨率会降低，因而就无法抓取屏幕的右边和底部近一半的地方。

在注册时需要一个电子邮件地址。

 

Jing只支持抓取矩形屏幕。但是，它也能在所选区域中抓取操作视频。它不支持延时抓取、热键、自动保存及滚动窗口等功能。可输出到剪贴板、文件和Screencast.com网站。它包括了一个图像编辑器，但没有视频编辑器。

需要到Screencast.com进行注册。
