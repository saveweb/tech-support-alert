简介

自从谷歌公司发布了针对智能手机的安卓操作系统2.x系列版本（包括代号松饼的2.1版、代号冻酸奶的2.2版和代号姜饼的2.3版），以及针对平板电脑的版本3（代号蜂巢）以来，该系统正快速发展。它在最新的版本4（代号冰激凌三明治，即ICS）上达到了另一个伟大的里程碑，它是既适合于运行在智能手机上，也适合运行在平板设备上的通用系统。
（译注：在上传本译文时，谷歌已经推出了4.1版本的安卓系统果冻豆Jelly Bean。）

如果你正在使用运行2.x版本安卓系统的智能手机，你会熟悉有四个硬件按钮——返回、菜单、主屏幕和搜索——你可以用它们与系统互动。谷歌公司的旗舰级智能手机Galaxy Nexus借用了内置于平板电脑的设计，它运行安卓4.0系统，现在在前面板上没有硬件按钮，一个也没有。这些硬件按钮变为虚拟按钮，并且该设备只保留了两个物理按钮——音量和电源按钮——分别位于设备的左右两侧。

正如屏幕截图所示，三个按钮返回、主屏幕和最近使用的应用程序经常出现在主屏幕上，而菜单按钮仅当某个应用程序需要它时才会出现。当你播放视频时，默认的媒体应用程序会隐藏屏幕上的所有这些按钮，并占用整个屏幕区域来播放你的视频。这是个相当巧妙的设计。

如果你得到了一台新的安卓设备，并且它运行着4.0版本的系统，那么这些技巧和提示将会对你有所帮助。为便于参考，我们也补充了一些关于较老版本安卓系统的相关信息（如果适用的话）。

注意：这些技巧和提示的效果在Galaxy Nexus的4.0版安卓系统上最好。如果你的安卓系统是前代的版本，或者被设备提供商修改过，那么某些步骤就可能与这里的不一样。（译注：由于没有Galaxy Nexus实物，所以下述有关安卓系统界面的用语可能与实际的系统有差别，请读者谅解。）

如何在打开的应用程序之间切换

当运行某个应用程序时，你只要轻按主屏幕图标，就能将该程序最小化。要返回该程序，其办法不是像在Windows中那样，点击任务栏。但安卓系统的做法也是相当直接的。

  1. 轻按“最近应用程序”图标，以列示出你最近打开的应用程序。(注释2)
  2. 向下滑动，就可以看到更多最近打开的应用程序。
  3. 从该列表中轻按某个应用程序的屏幕快照，就能立即切换到该应用程序。

注释1: 你可以把应用程序从最近列表中清除出去，办法是把它们一个接一个地向侧面滑移出去。
注释2: 在2.x版的安卓系统上，按住主屏幕按钮，就能列示出最近打开的应用程序。

 

如何关闭正在运行的应用程序

在安卓系统中，你不必为每个应用程序的运行实例做关闭操作。但在某些情况下，你可能需要强行停止某个运行不正常的应用程序，或者要开始玩游戏，那你就得真正关闭应用程序，以便开始进入一个新的游戏关卡，而不是继续运行上一个关卡。下面是如何操作的步骤：

  1. 运行“设置”程序，并轻按“设备”下的“应用程序”。
  2. 选择你想要关闭的应用程序。
  3. 轻按“强行终止”按钮，并再轻按“确定”按钮以确认关闭操作。

 

如何设置或取消打开某类文件的默认应用程序

在安卓系统中，你可以设置在默认情况下，某一类型文件由某个特定的程序打开，就像你在Windows下所做的那样。例如，如果你已经安装了如QuickPic之类的图像查看器，那么当你试图打开某个图像文件时，系统就会询问你选择某个应用程序（请参见屏幕截图）。

  • 要设置默认使用某个应用程序来打开文件，请在你轻按那个应用程序之前，勾选“在默认情况下使用此动作”。

当你下一次再轻按以打开某个图像文件时，它就总会用你已经选择的那个应用程序来打开文件，而不会再一次问你。

如果你需要取消默认应用程序的设置，该怎么办？

  1. 运行“设置”程序，并轻按“设备”下的“应用程序”项。
  2. 选择你想取消的默认应用程序。
  3. 轻按“默认启动”下的“清除默认”。

 

如何卸载应用程序

把应用程序从谷歌Play商店安装到你的安卓设备上是很方便的，类似的，卸载某个应用程序也是相当直截了当的。

  1. 轻按托盘上的“所有应用程序”图标，再选择顶端的“应用程序”标签页。
  2. 把你想要卸载的应用程序图标按住不放。
  3. 把该应用程序的图标拖放到“卸载（Uninstall）”图标上，然后轻按“确定”以确认此操作。（只有对于可以从系统中卸载的应用程序来说，才会出现“卸载”图标。）

另一种办法是，你可以运行“设置”程序，轻按“设备”下的“应用程序”，然后从列表上选择你想要卸载的应用程序。

 

如何向主屏幕上添加图标，或从上面删除图标

当你安装某个应用程序时，系统经常就会把应用程序的图标添加到你的主屏幕上。该图标可作为运行那个应用程序的快捷方式。当你需要亲手添加某个应用程序或桌面小部件（widget）的图标时，这个过程也是相当容易的。

  1. 轻按托盘上的“所有应用程序”图标。
  2. 选择顶部的“应用程序”或“小部件（Widgets）”标签页。
  3. 把你要要创建快捷方式的图标按住不放。
  4. 再把该图标拖放到你想要放置的那个主屏幕上。

从你的主屏幕上删除某个图标：

  1. 把应用程序、小部件或快捷方式的图标按住不放，直到屏幕顶部出现“删除(Remove)”图标为止。
  2. 把按住的图标拖放到“删除”图标上。

注释: 在2.x版安卓系统中，你还可以像下面这样添加快捷方式图标：把主屏幕的空白区域按住不放，或者按“菜单”然后轻按“添加”；在3.0版的安卓系统上，触碰主屏幕右上方的“定制按钮”，也可以添加快捷方式图标。

 

如何重新排放图标

当你在安卓系统上安装了新的应用程序时，系统会在没有请你选择某个主屏幕的情况下，就自动堆放图标。但你在这之后可以在主屏幕和托盘上重新排放图标。

  1. 把你屏幕上的图标按住不放，保持1秒的时间。
  2. 把图标拖到屏幕上的另一个角落，或者拖到或拖出托盘都可以，只是不能对永久保留的“所有应用程序（All Apps）”图标这样做。
  3. 你也可以把图标拖动到屏幕的左边或右边，直到屏幕开始滑动为止，然后把图标放到另一个屏幕上。

 

如何创建、重命名和删除文件夹

如果你有许多图标堆放在数个主屏幕上，那么与寻找放在托盘上的应用程序不同，你有时会难以快速找到想要打开的应用程序。那为什么不创建一个文件夹来把类似的应用程序分组放在一起呢？其过程比你能想得到的要简单得多。

  1. 把任一个图标按住不放，保持1秒。
  2. 把该图标拖放到另一个图标上方。接着系统就会自动把两个应用程序图标分组放在一个文件夹里。
  3. 触碰该文件夹，然后轻按“未命名的文件夹（Unnamed Folder）”，接着输入文件夹的名字，再轻按“完成”。

你可以在任何时候重命名某个文件夹——轻按某个文件夹，然后轻按文件夹名字来将它重命名。

系统会为你自动创建文件夹，并且它也会自动删除文件夹——轻按某个文件夹来展开它，然后清除文件夹内所有分组在一起的项目——只要按住不放，然后把它们一个接一个地从文件夹中拖出来，这样就行了。

注释: 老版的安卓系统不支持这个功能，但是你可以借助如GO Launcher EX之类的第三方应用程序来达到几乎相同的效果。

 

如何利用连续空格键来更快地输入

在输入的每句话结束时，除了可以键入句点再加上一个空格外，你可以轻按空格键两次，也能达到同样的效果。

 

如何输入特殊字符

如果你经常需要在屏幕里的字母和数字键盘之间切换，而这样做仅仅是为了键入句子中的逗号，那么为什么不看看下面的技巧以发现更多的实用内容呢？

当在字母键盘的状态下时：

  • , (逗号) => 把句点键(.)按住不放
  • ' (单引号) => 把句点键按住不放，并滑动以选择单引号(')
  • " (双引号) => 把句点键按住不放，并滑动以选择双引号(")
  • 切换大小写字母的开关 => 轻按两下Shift键来锁定（打开大写字母功能），轻按一下Shift键来解锁（关闭大写字母功能）。
  • 你想输入不在键盘上的特殊字母？ => 把某个键按住不放，然后滑动以选择一种字母变体（请参见屏幕截图）。
  • 仅需要输入数字键盘上的某个数字或符号？ => 触碰?123键，并滑动到相应的数字或符号键，然后你就会自动返回字母键盘。

当在数字键盘上时:

  • 所要输入的符号不在键盘上？ => 把某个键按住不放，然后滑动以选择某个变体，或者轻按#\<键以找到更多的符号。
  • 经常要输入等式？ => 把#\<键按住不放，再轻按某个符号，然后当你放开#\<键时，你就又回到数字键盘了。
  • 仅需要从字母键盘上键入某个字母？ => 触碰ABC键，并滑动到某个字母键，你就会自动返回到数字键盘。

 

如何剪切、复制和粘贴？

这些操作看起来很简单，但当你使用不同的应用程序时，有时其中的技巧会是很微妙的。通常，在使用电子邮件程序和其它一些应用程序时，会用到这些步骤：

  1. 要选择某个单词时，把它轻按两下或按住不放。
  2. 如果需要的话，拖动选择手柄，来调整选择区域，然后轻按“剪切”或“复制”。
  3. 轻按某个插入点，接着如果需要的话，再拖动选择手柄来调整所选区域。
  4. 选择“粘贴”命令（提示：如果“粘贴”按钮不见了的话，拨动一下选择手柄。）

注释: 当你在使用如安卓内置浏览器等的网络浏览器时，请使用按住不放的办法，来选择某个单词，因为“轻按两下”这个手势已经被用来缩放网页了。

 

如何关闭自动更正的功能

当你在持续输入文字时，系统会检查你的拼写并且给出一些建议使用的单词。此时当你键入一个空格、标点符号或回车符，以完成单词的输入时，除非你选择了其中的一个单词，否则那个高亮的建议单词会覆盖你输入的单词。如果你觉得自动更正功能很讨厌，你可以关闭该功能。

  1. 打开“设置”程序，并选择“个人”下的“语言和输入”。
  2. 轻按“安卓键盘”边上的“快速设置”图标。
  3. 选择“文字更正”下的“自动更正”。
  4. 选择“关（off）”掉“自动更正”功能。

注释: 另一种绕开自动更正功能的办法是，每当系统自动更正后的某个单词不是你想要的时候，你可以在自动更正动作发生后立即轻按“删除”键，然后系统就会返回用你原先所键入的单词。

 

如何锁定屏幕方向

当你运行某些特定的应用程序时，安卓系统的屏幕会对你握持设备的方向做出反应，相应地切换为纵向或横向视图。如果我想要把屏幕的显示方式锁定为某种视图，以使屏幕不会旋转为另一种视图，那该怎么办呢？很容易。

  1. 运行“设置”程序。
  2. 轻按“设备”下的“显示”。
  3. 去掉“自动旋转屏幕”的选项。

这样你的屏幕就会锁定为纵向模式。如果要解锁，就勾选上述的选项。

 

如何做屏幕截图

当你需要对你的安卓设备屏幕截图时，哪里有“打印屏幕（PrintScreen）”的按钮呢？使用一个按钮是无法奏效的，而是要组合使用两个按钮：

  1. 把电源和音量减小的按钮同时按住不放。
  2. 打开“画廊”应用程序，你就会看到你做的屏幕截图已经保存在“屏幕快照”相册中了。

注释: 旧版的安卓系统不支持此功能，但它有一种更复杂的屏幕截图方法，该方法不用root设备，但要像这样子来使用安卓的SDK。

 

如何选定安卓系统的壁纸

你可以轻松地从你系统上的那些壁纸中选择一张，并把它设定为你主屏幕的壁纸。

  1. 在主屏幕的某块空白区域上按住不放。
  2. 从弹出菜单中选择“画廊”、“动态壁纸”或“壁纸”。
  3. 选择某个图片并把它设为壁纸。

另一种方法是，你可以运行“设置”程序，然后到“设备 > 显示 > 壁纸”下，接着照上述的步骤2继续下去。

 

如何使用数据访问通知的功能

在你屏幕的顶部，你会看到通知消息，它会报告日历事件、警报、收到的邮件和其它正在进行中的事件。用手从屏幕的顶端向下滑动，就会显示通知消息的列表，然后从O-标记处向上滑动，就可以把它藏起来。

  • 依赖于通知消息的类型，你要把通知消息向侧边滑动来删除单条消息，或者轻按X-标记来把所有的通知消息从列表中删除。
  • 如果要访问一条通知，你只要轻按它，然后系统就会把你带到相关的电子邮件、事件提醒程序、消息或其它应用程序。然后如果可以的话，那条通知就会从列表中自动消掉。

注释: 当你的手指想要从屏幕顶部向下滑动时，你也要看看这个“快速设置”图标。触碰这个图标相当于运行“设置”程序。

 

如何在安卓默认浏览器中编辑书签和文件夹

就像大多数浏览器那样，安卓系统的默认浏览器让你能轻松地添加书签和文件夹，使你在浏览时能快速地到达相关的网站。

添加书签/文件夹

  1. 在你喜欢加为书签的网页上，轻按菜单按钮，并选择“保存到书签”。
  2. 如果需要，就修改书签的标题。
  3. 点击“确定”以保存书签，或者选择“加入到”，再点“其它文件夹”，以浏览到某个文件夹，或者是添加一个新的文件夹，然后轻按“确定”。

删除书签/文件夹

  1. 轻按“标签页”按钮，然后再点“书签”按钮。
  2. 把某个书签或文件夹按住不放，然后从弹出菜单中选择“删除书签”或者“删除文件夹”。
  3. 按“确定”来确认删除操作。

 

如何在主屏幕上添加网页快捷方式图标

如果你经常访问某个网页，你就会在主屏幕上轻按网页书捷方式或书签图标，来快速访问该网页，而无需触碰“浏览器”图标。你可以很容易地这种网页快捷方式图标添加到你的主屏幕上：

  1. 在你经常访问的网页上，轻按安卓浏览器上的“菜单”按钮。
  2. 选择“保存到书签”。
  3. 如果必要的话，就编辑快捷方式的标题，然后选择“添加到”，并轻按“主屏幕”。

下次你要访问同一个网页时，就只要轻按你主屏幕上的网页快捷方式图标即可。某些网站，包括Gizmo免费软件站点在内，已经拥有了为你定制好的网页快捷方式图标（请见屏幕截图）。

 

如何发送即时消息

除了第三方的应用程序外，由谷歌公司开发的Messenger可以让你作为安卓用户，来通过Wi-Fi、3G或4G无线网络使用实时的即时通讯服务。如果你要使用该服务并从你的安卓设备上发送即时消息，这个过程是相当直接了当的。

  1. 轻按Messenger应用程序图标。
  2. 轻按加号并从你的圈子中添加一个人或一组人。
  3. 添加文字或相片，并轻按“发送”按钮。

注释: Messenger程序或者是作为你安卓系统随带的独立应用程序，或者是作为Google+应用程序的一个组件，如果是后者的话，它可以从谷歌在线Play商店下载到。

 

如何设置电子邮件账号和同步选项

安卓系统可以添加多个电子邮件账户，包括Gmail、Exchange、雅虎和Hotmail等等。设置电子邮件账户的过程是相当简洁的，特别是对于下面的GMail例子而言。

  1. 打开“电子邮件”应用程序。
  2. 输入电子邮件地址和密码，然后轻按“下一步”进行验证。
  3. 勾选上“同步本账户的电子邮件”，并按你的喜好选择其它选项，然后轻按“下一步”。
  4. 输入你要在发出邮件上显示的名字，并轻按“下一步”以完成设置。

在这个设置完成后，你就可以打开“电子邮件”应用程序，来发送邮件并阅读收到的邮件，该程序会通过无线网络自动与你的GMail账户保持同步。

 

如何添加电子邮件签名档

当你使用“电子邮件”应用程序来发送邮件时，你可以设定邮件的签名档，如“该邮件是从我的移动设备上发送的”，这些文字会被自动添加到你邮件的结尾处。

  1. 打开“电子邮件”应用程序，轻按“菜单”按钮，然后按“设置”。
  2. 轻按你已经设置好的电子邮件账户。
  3. 轻按“签名”，再按自已的喜好输入一些字作为签名档，再轻按“确定”进行确认。

 

如何同步电子邮件、通讯录和日历

安卓系统让你能同时用多个账户处理和同步你的电子邮件、通讯录、日历和其它项目。你可以设置你的账户来自动或手工地在你的设备和服务器之间同步这些项目。

  1. 运行“设置”应用程序，选择“个人设置”下的“账户和同步”。
  2. 在最上方，轻按“账户和同步”，“开”即是自动同步，“关”即是手工同步。

当“账户和同步”处于关闭状态时，你可以触击账户，并手工同步通讯录或其它项目——或者是触击某个项目来进行同步，或者是触击“菜单”按钮，然后再点“现在就同步”，来同步所有项目。在每个项目下方会显示最近一次同步的日期和时间。

 

如何在你的移动设备上浏览和编辑文件

你可能在你的移动设备上有许多自己的文件，你想要对它们进行浏览、复制、重命名或删除等操作，就像你在Windows中所做的那样。但遗憾的是，你无法做到很多操作内容，因为4.0版的安卓没有内置的文件管理器，但你可以从谷歌在线Play商店轻松地安装第三方的应用程序。

  1. 运行“Play商店”应用程序。
  2. 轻按“搜索”按钮，并输入某个文件管理器的名字，如ES File Explorer，它是最流行的文件管理器之一。
  3. 轻按清单上的应用程序，并下载之。
  4. 打开所下载和安装的应用程序，并开始在你的移动设备上处理你的文件和文件夹。

 

如何在你的移动设备和个人电脑之间复制文件

如果你需要在你的移动设备和个人电脑之间复制文件和文件夹，那该怎么办？你可以使用所提供的USB线来连接你的移动设备和个人电脑，然后就可以了。更好的做法是，你可以通过本地的WiFi网络以无线方式连接这些设备，来做同样的事情，或者做出更多名堂，而无需连接线缆。

  1. 运行“Play商店”应用程序。
  2. 轻按“搜索”按钮，并输入某个远程桌面应用程序的名字，如AirDroid（这里有对该软件的评论）。
  3. 轻按搜索列表上的应用程序，并下载之。
  4. 打开该应用程序，并开始用它来连接你的移动设备到个人电脑上，以便用无线方式复制文件，或者在两个平台之间做更多的事情。

 

如何共享移动数据连接

当你启用移动数据连接来使用某台安卓设备时，如果另一台电脑设备如基于Windows的笔记本电脑能够共享使用该连接来访问互联网，这难道不是很方便吗？是的，你可以使用一台便携的无线热点来做无线连接，或者使用USB线缆来做有线连接。

把你的安卓移动设备变为便携的Wi-Fi无线热点：:

  1. 在你的移动设备上，进入“设置 > 更多 > 网络分享和便携式热点”。
  2. 轻按“便携式Wi-Fi热点”，以启用之。
  3. 轻按“配置Wi-Fi热点”，再输入ID和密码，以防止未授权而使用该连接。
  4. 在一台有Wi-Fi连接能力的电脑上，点击系统托盘上的网络图标，以识别和连接上你所设置的无线网络。

在你的安卓移动设备上启用USB网络共享:

  1. 用USB线把电脑连接到你的安卓设备。
  2. 在你的移动设备上，进入“设置 > 更多 > 网络分享和便携式热点”。
  3. 轻按“USB网络分享”来启用之。
  4. 在电脑上，点击系统托盘上的网络图标，以识别和连接到有线网络。Windows系统会花1分钟多的时间来发现可访问互联网的网络。

注释: 你也可以在你的安卓移动设备上启用蓝牙网络分享，并在该设备与电脑配对成功后，让电脑可以访问互联网。

 

如何备份和恢复你的数据

你在使用安卓系统时，系统会通过你在谷歌服务器上的账号，自动备份你的数据，如你已经安装的应用程序清单，你加入字典的单词，你的Wi-Fi无线网络密码，浏览器书签和其它设置。

  1. 运行“设置”程序，轻按“个人”下的“备份和重置”。
  2. 如果尚未勾选“备份我的数据”和“自动恢复”两个选项的话，就勾选上它们。

当你需要利用备份服务，来重置你的设备或重新安装应用程序时，你就可以从你的谷歌账户上恢复你的数据和设置。

 

如何锁定你的安卓设备

你是否不想让你的孩子胡乱摆弄你的手机，或者不想让别人阅读你设备上的电子邮件？你可以用密码、pin、图案或甚至用脸部识别来锁住它。

  1. 打开“设置”程序，并选择“个人”项下的“安全”。
  2. 选择“屏幕安全”项下的“屏幕锁定”。
  3. 轻按某种锁定类型，如图案、Pin、密码或人脸解锁，来锁定你的设备。

以后每当你开启设备时，你就都得用相同类型的安全方式来访问和使用你的设备。

 

如何重启系统

当你的安卓程序死锁或者系统变得没有反应时，你可以按照下面的方法重启系统：

  1. 把你设备的电源按钮按住不放，直到弹出菜单。再选择“关闭电源”，并轻按“确定”来确认此操作。系统就会关机。
  2. 再把电源按钮按住不放，直到出现谷歌的logo标志为止，这样系统就打开了。
