简介

我是个十足的数码摄影爱好者。我渴望照出完美的照片，让主题、构图、光线和颜色都能恰到好处。然而事与愿违，大多数时候都需要我们做一点点修改，才能使照片看起来像我们希望的那样。网上免费的图像编辑与照片修正软件简直是不胜枚举，要从中找到最适合自己的那个实在是让人想想就头大。在评估比较了30多个软件后，我把范围进一步缩小，并归分成三类：初级、中级和高级工具。

初级类都是一些小巧的精品软件，能快捷方便地帮你对图像的整体光线、色彩和色调做细微的修正，而没有一堆乱七八糟的高级功能。这些软件也提供了裁剪、锐化以及去红眼这些基本功能。

中级类的软件提供了更为高级的功能，例如图层、添加说明文字与图形，以及仅选择图像的一部分进行修正等等。这些软件也提供滤镜以实现纹理、艺术效果、边缘增强、相框等。

高级类都是一些专业软件，可以匹敌Photoshop或是Paint Shop Pro。
 
讨论
初级编辑器

LightBox Image Editor是非常优秀的小工具，我仅仅使用了几分钟就爱上它了。它的界面很吸引人，既简洁又直接，通过一些滑动条来修改参数，以便直观地调整你的图像。软件还提供了分屏视图，帮助你更好地对比修改前与修改后的图像。它包括了一些简单的功能，诸如去红眼、锐化、裁剪、缩放、添加相框以及打印。我实在太喜欢LightBox了，所以决定升级到Pro版，以便获得一些额外功能，不过其免费版已经足够好了，值得一试。这是我首选的初级编辑器。

Photo! Editor是一款初级的照片修正软件，附带一些专业水准的功能。它具备所有基本的增强功能，例如调节亮度、对比度、饱和度、色偏，以及一些修饰或“化妆”功能，类似于Photoshop中神奇的修复画笔工具。记得要注意工具栏图标旁的下箭头，这将有助于你使用更有针对性的手动选项。
中级编辑器

PhotoFiltre是介于初级与中级之间的优秀软件，拥有一系列良好的特性。想要对照片进行简单的调整，只要点击工具栏上的按钮就能快速地完成。通过深入菜单选项，你可以完成更高级的修改。它有一些好用的功能，可以选取图像局部、进行简单的涂抹、模糊化、描绘基本形状，或是添加文本。PhotoFiltre也有一些精选的滤镜，以便为你的照片增加效果，而且支持批处理功能。

Paint.NET在所有中级照片修饰软件中是个非常不错的选择。它拥有一系列高级的照片修正功能，包括曲线与色阶。它实现了完善的图层功能，有混合模式以及可调节的透明度。它有着相当完善的选择、绘画和图形绘制功能。它也提供了一些不错的滤镜，能为图像增添特殊效果。总而言之，Paint.NET作为一款免费的图像修饰软件，功能已经相当多了。注意你的电脑需要先安装Microsoft .NET Framework 3.5 SP1(Windows 7中已自带)。

Artweaver：如果你就是不想安装.NET Framework，或者仅仅是想在Paint.NET之外多一种选择，你不妨可以看看这个。尽管它的目标更专注于绘画方面，它也带有一组很棒的图像校正功能。Artweaver的曲线和色阶功能在我看来甚至比Paint.NET还要好上那么一点。除此之外，二者的功能基本相当。
高级编辑器

GIMP据我了解是目前唯一配得上称作高级图像编辑器的免费软件。它的学习曲线比前面提到的软件都要更陡，然而功能自然强大得多。与大多数Windows程序不同，它拥有一个多窗口的界面。如果你还是图像编辑软件方面的新手的话，要从GIMP学起有些偏难了，所以我还是建议从以上介绍的初级或中级编辑器开始学习为好。
自成一派

Photoscape：它真的与上述提到的编辑器都不属于同一类，因为它的一些功能都是一般软件不关注的，称它为一个图像套件可能更为恰当。事实上是有好几位读者要求将这款软件包括进来，最后我还是答应了。一旦你适应了它那与众不同的界面，就能发现它可不单单是个照片编辑器。它包含了批处理函数功能、一组拼贴画工具、gif动画生成器、批量改名功能、raw图像转换器、截屏功能、图像分割功能、照片布局和打印功能，以及一些其他功能。编辑器本身相当稳健。虽然它缺少了区域选取、刷子和克隆风格这样基本的功能，但它拥有一些很赞的手动或自动颜色校正工具。它也有滤镜、添加边框、文字、形状、文本框以及其他与矢量对象所类似的，直到与图像接合或保存图像。它还包含了大约500个剪贴画可以用来放在你的照片中。尽管我个人还是偏向于上面提到的那些图像编辑器，但可能很多用户会发现Photoscape海量的功能更加方便，使用起来乐在其中。至少多数人都可以把它作为上述其他工具的辅助。
