Mejor Editor Gratuito de Etiquetas MP3

Introducción

MP3 se ha convertido en el formato de audio más común para reproducir, almacenar y transferir música. Si necesitas reproducir archivos MP3 y que se muestre correctamente toda la información de las etiquetas, o quieres organizar una gran colección de archivos MP3 en base a la información de sus etiquetas, te será muy útil un buen editor de etiquetas MP3.

En esta categoría hemos analizado tanto los editores de etiquetas MP3 independientes como los integrados en el reproductor, fijándonos en tres necesidades básicas, que son:

  • Soporte para etiquetas básicas, posibilidad de insertar carátulas y letras.
  • Renombrado de archivos usando la información de las etiquetas, así como edición de etiquetas usando los nombres de archivo.
  • Búsqueda en bases de datos online y posibilidad de guardar esa información en las etiquetas de los archivos MP3.
Editores de Etiquetas Independientes

TagScanner es mi favorito. Admite etiquetas báscicas, como artista, álbum, título, pista, año y género, y también etiquetas avanzadas, incluidas las de compositor, grupo, BPM (“beats per minute” o “pulsos por minuto”), etc. Muestra las letras de las canciones debajo de la carátula de portada.

La pestaña Music Renamer permite renombrar archivos y generar etiquetas a partir de los nombres de archivo.

Otra de las pestañas de TagScanner es Tag Processor, que permite visualizar las etiquetas y carátulas  de las bases de datos en línea antes de incorporarlas a tus archivos MP3.

Sin embargo, el menú del programa no es el habitual de ventanas. Quizá necesites algo de tiempo para familiarizarte con su interfaz de usuario.

Mp3tag es una buena opción si prefieres usar una interfaz más normal, que incluya un menú con ventanas. Lo bueno de Mp3tag es su diseño sencillo y limpio, con una herramienta muy potente y a la vez fácil de utilizar para editar las etiquetas de los archivos MP3.

Las etiquetas básicas se muestran en un panel frontal personalizable. Las etiquetas avanzadas se muestran en una ventana aparte que incluye todas las etiquetas existentes almacenadas en tus archivos MP3. Si una determinada etiqueta para agregar las letras de las canciones o cualquier otra información no existe, la puedes crear. O si una etiqueta está repetida, se puede eliminar fácilmente.

Mp3tag te permite renombrar archivos en base a la información de sus etiquetas, importar o exportar información de sus etiquetas y crear listas de reproducción. Trabaja con bases de datos online como Amazon, freedb, MusicBrainz y otras bases de datos de búsqueda, de forma que puede etiquetar automáticamente archivos e incrustar varias imágenes de portadas de discos.

También tiene una potente Plataforma de Fuentes Web, lo que hace posible obtener información del álbum de muy distintas bases de datos en línea. Las fuentes web más recientes están disponibles en los Foros de Mp3tag, en el apartado Web Sources Archive. Puedes descargar las que necesites y extraerlas en la carpeta “data” de la ruta en la que tengas instalado Mp3tag. De esa forma amplías el acceso a las bases de datos online para el etiquetado automático.

Kid3 (acrónimo de KDE ID3) es un editor de etiquetas de audio para Linux que permite editar todas las etiquetas de los archivos MP3. Puedes generar etiquetas a partir de los nombres de archivo y viceversa, y también importar etiquetas desde bases de datos online, incluidas gnudb.org, TrackType.org, Discogs y MusicBrainz.

Esta aplicación permite arrastrar y soltar una portada para añadirla al archivo correspondiente y ayuda a buscar carátulas de álbumes y letras de canciones a través del navegador, aprovechando así todos los recursos en línea. Otras funciones son las de convertir entre etiquetas ID3v1.1, ID3v2.3 e ID3v2.4, editar etiquetas de varios archivos a la vez, generar listas de reproducción, renombrar y crear directorios a partir de las etiquetas, cambiar entre mayúsculas y minúsculas y exportar etiquetas.

A pesar de todas estas características, le falta una vista en columnas accesible para otros editores, de forma que se puedan localizar de un vistazo los campos de una misma etiqueta que tienen diferentes valores en varios archivos. Debido a esta carencia, sólo verás el símbolo de igual tachado o "distinto" en las etiquetas en caso de que tengan diferentes valores y estén seleccionados varios archivos.

MusicBrainz Picard funciona en múltiples plataformas y adopta un nuevo concepto, el de las etiquetas enfocadas al álbum, diferente del actual con etiquetas dirigidas a las pistas o a los archivos.

El programa identifica la información de la pista comparando la huella digital única de ese archivo con la información almacenada en la base de datos de MusicBrainz. Una vez emparejada la pista, puedes salvar su correspondiente información de etiqueta en los archivos MP3 seleccionados. How To Tag Files With Picard (Cómo Etiquetar Archivos Con Picard, en inglés) es una guía de inicio rápido muy útil.

Al igual que con Mp3tag, al comprobar los detalles del archivo seleccionado presenta una lista de todas las etiquetas existentes, incluidas las letras, que luego puedes editar. Si una determinada etiqueta aún no existe, puedes añadirla en cualquier momento.

Pero en la versión analizada no se podían editar las carátulas de portada salvo que las actualizase en las bases de datos online.

MetatOGGer introduce espacios de trabajo para hacer más fácil el proceso de etiquetado. Estos espacios de trabajo se pueden configurar, guardar y cargar.

El programa cuenta con un panel desplegable situado a la izquierda, para ver o editar las etiquetas y portadas del archivo seleccionado. Hay otro panel desplegable a la derecha para buscar carátulas o importar etiquetas de su base de datos.

Además del etiquetado con scripts y a partir de nombres de archivo, admite el etiquetado basado en la huella acústica, función que me pareció muy precisa. Las letras de las canciones se pueden añadir a la etiqueta si se desea y si se pueden extraer de Wikia.com.

Como punto negativo, este editor sólo admite una carátula por archivo y la búsqueda de etiquetas en su base de datos no parece ser muy estable.

MPTagThat es un editor de etiquetas de código abierto relativamente novedoso. Además de la edición de etiquetas, el programa incluye algunas funciones extra, como reproducción de audio, extracción (ripeado), conversión y grabación de archivos de música.

En cuanto al etiquetado, permite etiquetar automáticamente a partir del nombre del archivo o de Internet, así como la identificación de archivos. Otra función muy útil es que puede añadir automáticamente a la etiqueta las letras de las canciones y las ilustraciones de portada sacadas de Internet, si bien tarda un poco en encontrar imágenes.

El programa tiene más características, como organizar o renombrar archivos a partir de la información de las etiquetas, convertir en mayúsculas y minúsculas, eliminar etiquetas, ejecutar scripts, etc.

Esta aplicación sigue desarrollándose activamente y tiene potencial para mejorar su calidad. No te sorprendas si encuentras algunos errores de manejo en la versión actual, que es la que hemos analizado.

A diferencia del programa anterior, The GodFather es un editor de etiquetas con una biblioteca integrada para organizar archivos MP3 basándose en la información de las etiquetas. El programa explora tus carpetas con archivos MP3 y automáticamente te sugiere a qué archivos cambiaría el nombre.

En el apartado “biblioteca” (library), puedes editar y mostrar hasta cuatro imágenes por álbum almacenadas en archivos MP3. Además, la función “Buscar letras usando Google” muestra los resultados en la página de búsqueda de Google o en la página que tenga las letras de los archivos seleccionados. Esto acelera la búsqueda de letras, pero tendrás que copiarlas y guardarlas en la etiqueta manualmente, por lo que no es tan eficiente como MPTagThat.

Además de la edición manual de etiquetas, permite el etiquetado  automático a través de bases de datos online como freedb, Allmusic y Amazon. Sin embargo, All Music Guide está deshabilitada por un posible incumplimiento de las condiciones del servicio, y la función integrada de búsqueda en Amazon encuentra menos resultados que los demás editores de etiquetas analizados. Otro inconveniente es que este programa no admite caracteres Unicode.
Editores de Etiquetas Integrados

MusicBee es un reproductor y organizador de música que incluye un editor de etiquetas tan bueno como uno independiente.

Permite etiquetar automáticamente álbumes o pistas, buscar ilustraciones, letras y otra información de etiquetas en la web, guardar etiquetas e incrustar letras y múltiples imágenes en los archivos de música de tu colección. El monótono trabajo de etiquetar es mucho más fácil e intuitivo gracias a este programa.

Como alternativa, MediaMonkey es también un reproductor de música con un editor de etiquetas integrado que cubre las necesidades básicas. Permite editar la información de etiquetas básicas y detalladas, etiquetado automático a partir de nombres de archivo, mover o renombrar archivos a partir de las etiquetas de pista.

También permite editar letras de canciones y múltiples imágenes con tipos y descripciones de las ilustraciones del álbum, fijar etiquetas en los archivos MP3 seleccionados con búsqueda automática, etiquetar la carátula del álbum y añadir cualquier otra información de la base de datos de Amazon.

AudioShell no es una aplicación independiente por sí misma sino una extensión de la shell de Windows Explorer. A diferencia del resto de aplicaciones mencionadas en este artículo, permite ver y editar información de las etiquetas directamente en el Explorador de Windows.

Una vez instalado AudioShell, no tienes que volver a ejecutarlo. Añade 27 columnas adicionales para mostrar etiquetas en la vista “Detalles” del Explorador.

Agrega varias pestañas a la ventana “Propiedades” del archivo MP3. En la pestaña “AudioShell Tag Editor” puedes editar etiquetas báscicas y carátulas de portada. La edición de las letras de canciones se hace en la pestaña “Resumen”, en “Opciones avanzadas”. Se pueden editar grupos de etiquetas seleccionando varios archivos.

AudioShell no ofrece conexión con bases de datos online, pero es un buen programa para complementar cualquiera de las aplicaciones anteriores.
 
Otros Editores de Etiquetas
Éstos son otros editores de etiquetas MP3 que se han mencionado en los comentarios o vistos en otras fuentes de información. Al no estar clasificados en este análisis, os ofrezco un listado con breves descripciones y enlaces a los sitios correspondientes para facilitar la consulta.

  • MP3 Book Helper permite renombrar archivos y editar etiquetas por variables o por expresiones regulares. Las imágenes y letras se muestran en ventanas separadas con una búsqueda vinculada a Allmusic o a Google en el navegador. La búsqueda de letras redirige a la página de búsqueda de Google relacionada con el archivo seleccionado, pero la búsqueda en la base de datos de freedb para obtener información de etiquetas da menos resultados válidos que otros editores.

  • ID3-TagIT soporta un montón de información de etiquetas, incluidas letras y numerosas imágenes de álbumes. También trabaja con la base de datos de freedb, pero el ratio de coincidencias en los resultados es bajo y a veces da errores de 'excepción no controlada'. En algunos arcivos MP3 muestra las letras sin saltos de línea.

  • EasyTag es un editor multiplataforma para etiquetas ID3 de diferentes archivos de audio que usa una interfaz GTK+. Hay que destacar que no admite etiquetas de letras de canciones.

  • Mp3 Tag Tools es un pequeño programa para editar etiquetas básicas y letras de canciones e incrustar varias imágenes relacionadas con el álbum en los archivos MP3, pero desgraciadamente no permite incorporar las etiquetas de las bases de datos online, una de las necesidades que hemos considerado básicas en este análisis.

  • Jaangle (antes Teen Spirit) es un reproductor y organizador de música con un editor de etiquetas integrado. El buscador en Google Lyrics es potente en la búsqueda automática de letras, pero al escribirlas en los archivos da errores. Tampoco trabaja bien añadiendo ilustraciones de álbum al archivo, tal y como hemos comprobado con otros editores de etiquetas. Además de lo anterior, otras funciones de edición de etiquetas no son mejores que las de MediaMonkey.

  • Zortam Mp3 Media Studio es un organizador de MP3 multifunción con un editor de etiquetas integrado. Permite editar etiquetas, incluidas las de letras de canciones y múltiples imágenes, y permite actualizar las etiquetas online a través de Amazon. Sin embargo, el acceso ilimitado a la base de datos online de Zortam Music para letras de canciones y carátulas de portada, así como otras caracterísitcas básicas de edición, sólo se pueden usar en la versión Pro, de pago.

  • Abyssmedia ID3 Tag Editor ofrece funciones muy básicas de edición de etiquetas, sin incluir letras o carátulas de portada. No hace búsquedas online para obtener información de etiquetas.
